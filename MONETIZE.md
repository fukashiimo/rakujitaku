# 収益化ガイド

このアプリはチェックリストの持ち物に「Amazonで探す」「楽天で探す」リンクを表示します。
アフィリエイトIDを設定すると、リンク経由の購入が成果報酬になります。

IDの貼り付け場所は [app.js](app.js) の冒頭にある `affiliateConfig` です。

```js
const affiliateConfig = {
  amazonTag: "",   // ← Amazonアソシエイトのトラッキングid
  rakutenId: "",   // ← 楽天アフィリエイトid
};
```

IDが空のあいだも、リンクは普通の検索ページとして動くのでアプリは壊れません。

---

## 1. 楽天アフィリエイト（審査なし・最短でここから）

1. https://affiliate.rakuten.co.jp/ に楽天会員IDでログイン
2. 登録情報（サイトURL）に公開URLを記入: `https://fukashiimo.github.io/rakujitaku/`
3. 管理画面の「リンク作成」で任意のリンクを作ると、URL内に
   `hb.afl.rakuten.co.jp/hgc/【ここがアフィリエイトID】/` の形でIDが含まれる
   （例: `1a2b3c4d.5e6f7g8h.1a2b3c4d.5e6f7g8h` のようなドット区切り英数字）
4. そのIDを `affiliateConfig.rakutenId` に貼り付けてコミット・プッシュ

- 報酬: 商品価格の2〜4%程度（ジャンルによる）
- 支払い: 原則楽天キャッシュ。月3,001円以上などの条件を満たすと銀行振込も選択可

## 2. Amazonアソシエイト（審査あり）

1. https://affiliate.amazon.co.jp/ でアカウント作成
2. サイトURLに `https://fukashiimo.github.io/rakujitaku/` を登録
3. 発行されるトラッキングID（例: `rakujitaku-22`）を `affiliateConfig.amazonTag` に設定
4. **注意: 登録から180日以内に3件の売上**が発生しないとアカウントが閉鎖される
   （再登録は可能。まず楽天で運用しつつ、アクセスが増えてから申請するのがおすすめ）

### 代替: もしもアフィリエイト経由

Amazon直接の審査が通らない場合は https://af.moshimo.com/ 経由でAmazonと提携する方法もあります。
ただし、もしも経由はリンク形式が異なる（`//af.moshimo.com/af/c/click?...` 形式）ため、
その場合は `buildAmazonUrl` の書き換えが必要です。必要になったら相談してください。

## 3. 法律面（対応済み）

2023年10月施行のステマ規制により、アフィリエイトリンクにはPR表記が必要です。
チェックリスト画面に「※アフィリエイト広告（PR）を含みます」の表記を実装済みです。

## 4. 収益を伸ばすには

このタイプのアプリの収益は **訪問者数 × リンククリック率 × 購入率** で決まります。
まずは訪問者を増やすのが最優先です。

- X（Twitter）やInstagramで旅行準備の文脈でシェアする
- 「旅行 持ち物 チェックリスト」等の検索流入を狙う（SEOメタタグは設定済み）
- 旅行系のまとめ記事やブログからリンクしてもらう

目安として、月1,000人が使って数%が購入すると月数百〜数千円規模。
大きく稼ぐより「自分のアプリが初めてお金を生む」体験をまず作るのが現実的です。
